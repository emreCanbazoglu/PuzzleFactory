# Feature Specification: Director Planning Gate

**Feature Branch**: `011-director-planning-gate`  
**Created**: 2026-03-11  
**Status**: Draft  
**Input**: User description: "The fusion director phase should be strong enough before ideation starts."

## Requirements *(mandatory)*

- **FR-001**: Each cell must generate a structured director plan before any concept ideation artifact is produced.
- **FR-002**: The director plan must include ranked variation targets with materially different unified verbs, pressure structures, or board-update logic.
- **FR-003**: Concept generation must consume a selected director-plan variation instead of relying on a generic pair summary only.
- **FR-004**: The pipeline must persist the director plan so a fresh session can resume ideation from the same pre-ideation state.
