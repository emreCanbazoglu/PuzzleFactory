# Feature Specification: Agent Role Hardening

**Feature Branch**: `006-agent-role-hardening`  
**Created**: 2026-03-11  
**Status**: Draft  
**Input**: User description: "Agents need more detailed approaches, rules, and a clearer director/designer handoff."

## User Scenarios & Testing *(mandatory)*

### User Story 1 - Detailed Role Guidance (Priority: P1)

As an operator, I need agent files to contain real working instructions instead of shallow descriptions.

### User Story 2 - Director-Led Fusion (Priority: P1)

As a reviewer, I need fusion ideas to be derived from the fusion director and designers rather than a human detail override.

## Requirements *(mandatory)*

- **FR-001**: Agent files MUST define process, rules, and quality bars.
- **FR-002**: Runtime MUST create a director brief before concept generation.
- **FR-003**: Designer stages MUST have access to the director brief.
