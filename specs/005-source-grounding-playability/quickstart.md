# Quickstart: Source Grounding and Playability

1. Run `python3 .specify/scripts/python/wave_runner.py runs/wave_001/run_config.json`
2. Open one generated concept card and confirm it names the two source games.
3. Open one generated `prototype.html` and verify:
   - clicking exposed screws changes the board
   - win is reachable
   - lose is reachable after wasting moves
