# Data Model: Evaluation and Decisioning

## ScoreInput

- `concept_id`
- metric scores (clarity, depth, level_scalability, production_feasibility, retention_potential, novelty)

## DecisionRecord

- `concept_id`
- `auto_score`
- `auto_decision`
- `final_decision`
- `mode`
- `override`
- `finalized_at`
- `rationale`
