## User Scenarios & Testing *(mandatory)*

### User Story 1 - Human Review Artifact (Priority: P1)

As a reviewer, I need a clear place to leave feedback on each candidate so the pipeline can incorporate human judgment.

### User Story 2 - Core Feeling Prototype (Priority: P1)

As a player, I need the generated prototype to deliver the core feeling of the fusion, not just demonstrate one interaction.

## Requirements *(mandatory)*

- **FR-001**: System MUST generate a human review artifact per cell.
- **FR-002**: `Pixel Flow! + Screw Jam` prototype MUST include conveyor, dock, box dispatch, layered screw exposure, win, and lose loop.
- **FR-003**: Prototype MUST include enough feedback and pacing to communicate a retry-worthy puzzle.
