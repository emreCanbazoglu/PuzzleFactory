# Quickstart: Game Library Chat Review

1. Run library validation:

```bash
python3 .specify/scripts/python/validate_game_library.py
```

2. Prepare a review brief for one game:

```bash
python3 .specify/scripts/python/prepare_game_library_review.py ios-6751056652-pixel-flow
```

3. Use the generated brief as the context for the human-agent review conversation.
